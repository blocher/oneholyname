== Description ==
42functions is a company that brings SEO (Search Engine Optimisation) and technique together. And therefore we are happy to introduce the Social Media Tracking plugin to you. The Social Media Tracking plugin extends your posts, pages, feeds etc. with social media buttons. And track the interaction with the buttons using your Google Analytics account.

Currently supported buttons:

* Facebook
* Twitter
* Google+ 

Upcomming features:

* Button placeholders, only load the real button when the visitor interacts with a post
* Button area that scrolls allongside your page

More info:

* More info on [Social Media Tracking](http://www.42functions.nl/plugin-smt [NL]).

== Installation ==

Download, Upgrading, Installation:

**Upgrade**

1. First deactivate the Social Media Tracking plugin
1. Remove the `xlii-social-media-tracking` directory

**Install**

1. Unzip the `xlii-social-media-tracking.zip` file. 
1. Upload the the `xlii-social-media-tracking` folder (not just the files in it!) to your `wp-contents/plugins` folder. If you're using FTP, use 'binary' mode.

**Activate**

1. In your WordPress administration, go to the Plugins page
1. Activate the Social Media Tracking plugin and a subpage for the Social Media Tracking plugin will appear
   in your Options menu.

If you find any bugs or have any ideas, please mail me.
